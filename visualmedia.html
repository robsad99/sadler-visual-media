Visual Media

Course Description

An introduction to the visual image and visual thinking applied to photography, video, and film. Sophomore standing required. For this class we have four projects. The first is a magazine proposal and spread to work on our design skills in InDesign. The second project was a photo project to work on our photography editing skills in Photoshop. The third project was a video report to work on our video editing skills in Premiere. Lastly, our last project is our web presence to work on our basic coding skills in Atom and GitHub.
